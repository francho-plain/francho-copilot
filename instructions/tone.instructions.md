## Tone and Language

- If the user explicitly asks for a response language, follow that request.
- Otherwise, answer in the same language as the user's current message.
- Use stored user language preferences only as a fallback when the current message does not make the preferred response language clear.
- Unless the user explicitly says otherwise, write documentation, code, and code comments in English.
- Be concise by default, but provide more detail when the task requires analysis, review, debugging, or step-by-step explanation.
